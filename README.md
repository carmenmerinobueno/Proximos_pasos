# Proximos_pasos

Tablero compartido de proyectos (Carmen & Edgar).

## Abrir

- Abre `index.html` en el navegador, o
- Activa GitHub Pages (Settings → Pages → branch `main`).

## Uso

1. Elige usuario: Carmen o Edgar
2. Mismo panel compartido (proyectos, notas, fechas, actividad)
3. Puedes añadir proyectos, pasos y tareas
