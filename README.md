# DHARMA SOLUTIONS · Tablero compartido

Tablero de proyectos de **Carmen** y **Edgar**, con sincronización en la nube (gratis).

## Link web (cuando actives GitHub Pages)

https://carmenmerinobueno.github.io/Proximos_pasos/

## Activar el link (solo una vez, en el PC)

1. Entra en: https://github.com/carmenmerinobueno/Proximos_pasos
2. Arriba: **Settings**
3. Menú izquierdo: **Pages**
4. En **Source** / **Branch**: elige `main` y carpeta `/ (root)`
5. Guarda (**Save**)
6. Espera 1–2 minutos y abre el link de arriba

## Ponerlo en el móvil (como una app)

### iPhone (Safari)
1. Abre el link en **Safari** (no Chrome)
2. Toca el botón **Compartir**
3. **Añadir a pantalla de inicio**
4. Confirma — te quedará el icono **DHARMA**

### Android (Chrome)
1. Abre el link en **Chrome**
2. Menú **⋮** → **Instalar app** o **Añadir a pantalla de inicio**
3. Confirma

## Cómo funciona la sync

- Al guardar (tarea, idea, diario…), se sube a la nube compartida
- El otro dispositivo lo recibe automáticamente (cada ~15 s) o al tocar **Sincronizar ahora** en Ajustes
- No hace falta código de pareja: los dos usáis el mismo link

## Uso

1. Abre el link (o la app del móvil)
2. Elige **Carmen** o **Edgar**
3. Trabajad con normalidad — ambos veis lo mismo
